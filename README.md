# LinkVault 🔗

An Instagram DM Resource Organizer. Never lose a PDF, link, or freebie from a creator's DM again.

## The Problem

You comment on an Instagram post. The creator sends you a free PDF, guide, or link in your DMs.
You save it somewhere. Two weeks later — you have no idea where it is.

LinkVault fixes that.

## What It Does

- Save any resource (PDF, link, video, tool) with one tap
- Tag it by creator, category, and topic
- Search and filter your entire vault instantly
- AI organizer suggests what to do with each resource
- Weekly digest of everything you saved

## Tech Stack

- Frontend: React
- Backend: Supabase
- AI: Claude API (Anthropic)
- Styling: Tailwind CSS

## Features

- Add resources by URL or file upload
- Tag by creator Instagram handle
- Categories: Finance, Marketing, AI Tools, Design, and more
- Favorites — star what matters most
- AI summary and recommended next action per resource
- Mobile-first design

## Getting Started

```bash
git clone https://github.com/tanishapra24/linkvault
cd linkvault
npm install
npm run dev
```
